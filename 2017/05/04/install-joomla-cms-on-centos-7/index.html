在 CentOS 7 中安装 Joomla! 的教程

发表于 2017-05-04 更新于 2020-09-07 分类于 website-building

目标：

  1. 在 CentOS-7-x86_64 系统中安装最新版本的 Joomla! CMS。
  2. 支持 PHP 7。

条件：

  1. CentOS 7 系统安装就绪。
  2. 普通用户帐号，拥有 sudo 权限。

步骤：

  1. 安装 LAMP 之 A (Apache)。
  2. 安装 LAMP 之 M (MariaDB)。
  3. 安装 LAMP 之 P (PHP)。
  4. 配置 MariaDB 数据库。
  5. 配置 Joomla! 程序。
  6. 配置防火墙。
  7. 安装 Joomla! 程序。
